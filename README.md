# Simple notifications for Vue.js
![](ezgif-7-7bbdd698e363.gif)

## Setup 
```console
npm i notifications-vue
```
## Project setup
```javascript
// Add in main.js file:

import NotificationsVue from 'notifications-vue'
Vue.use(NotificationsVue);
```
## Usage
```javascript
this.$notifi.new("Some message");
```
